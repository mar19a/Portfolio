"Seekers" - Real Estate Website

Completed on May 3, 2023, this project was a joint effort with Gary Dong for CS 411, Software Engineering, at Boston University. We created a real estate website named "Seekers" using three APIs: Zillow API, Local Business Data from RapidAPI, and News API.

The website offers home for sale lookups using the Zillow API, local business recommendations based on user preferences and city data, and Massachusetts news. Users can contact our agent's services or ask questions by submitting a form. We receive a copy of all user submissions at our email, mariano@bu.edu.

The website utilizes a MySQL database for user login storage and also includes Google Oauth as an additional user login option. The project has both a front end and a back end. A demo of the project can be found here.

The project repository can be cloned and set up with npm install for dependencies, npm install react-google-login --legacy-peer-deps for Google Oauth dependencies, and npm start to start the website. The backend server can be run with node server.js. XAMPP is also required.
